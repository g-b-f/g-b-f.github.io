pipeline.chk(task=None, min_verbose=1, *, timer=True)#

Displays status of python code.

Parameters:

  • task (str, optional) – String describing the current task. Passing None will end the current timer without starting a new one

  • min_verbosity (int, optional) – minimum verbosity level required to display

  • timer (bool, optional) – Whether to time the current task

Notes

Do chk() (i.e, with no input) at end of program, or before statements that will print.

Examples

>>> import time
>>> chk("sleeping for a long time")
>>> time.sleep(5)
>>> chk("sleeping for a short time")
>>> time.sleep(0.3)
>>> chk()
12:00:00 sleeping for a long time... done, 5.0 seconds.
12:00:05 sleeping for a short time... done, 300 milliseconds.
pipeline.cluster_prep(window_df, acbDict, kernel=range(0, 10))#

Prepares the data for clustering

Parameters:

  • window_df (DataFrame) – DataFrame containing patients, ID of prescription, and age at prescription

  • acbDict (dict) – Dictionary mapping drugs to scores

  • kernel (iterable, optional) – Kernel used for smoothing via linear convolution. Set to None to not smooth

Returns:

  • window_df (list) – List of scores for each patient, convolved using kernel

  • pat (numpy array) – List of patients, in same order as window_df

pipeline.clustering(window_df, pat, n_clusters, limit=None, metric='dtw', **kwargs)#

Creates time series clusters using tslearn.clustering.TimeSeriesKMeans()

Parameters:

  • window_df (iterable of np.array) – Scores for each patient

  • pat (np.array) – Patient IDs

  • n_clusters (int) – number of clusters

  • metric ({"euclidean", "dtw", "softdtw"}, optional) – Metric to be used for both cluster assignment and barycenter computation. If “dtw”, DBA is used for barycenter computation

  • limit (int, optional) – Use at most this many patients in calculation

Returns:

lab_df – DataFrame of patients, each labelled with their predicted cluster

Return type:

DataFrame

Notes

metric= “euclidean” does not work for datasets without equal-length time series, such as those used in this pipeline

kwargs are passed to tslearn.clustering.TimeSeriesKMeans().

pipeline.run_window(groups, tempDir, name='pipeline')#

Runs the C file

Parameters:

  • groups (groupBy object) – Patients grouped using DataFrame.groupby(‘patid’)

  • tempDir (str) – path to temporary directory

  • name (str) – argument for handling corner cases on non-unix systems. See notes below.

Returns:

windowOut – DataFrame created by the C function

Return type:

DataFrame

Notes

Resultant file will be saved in tempDir/slideWindow.csv

On non-unix systems, when this function is imported as a module (not top level), name must be set to __name__ or "__main__". Leaving it at the default value will not achieve this, as __name__ will resolve to the name of the library, not "__main__".
